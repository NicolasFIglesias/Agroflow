❓ Como usar tags nos modelos

O que são tags?

Tags são marcadores no formato {{NOME_DA_TAG}} que você coloca no seu arquivo Word. Quando gerar o contrato, o sistema substitui cada tag pelo valor real automaticamente.

Como criar o template

  1. Abra o Word e crie seu contrato normalmente
  2. Onde quiser inserir um valor dinâmico, escreva {{NOME_DA_TAG}}
  3. Importante: Selecione toda a tag e aplique um único estilo. Não troque a formatação no meio da tag (ex: parte em negrito, parte normal) — isso quebra o reconhecimento
  4. Salve como .docx e faça upload aqui

Erro "Multi error" ao baixar?

Isso ocorre quando uma tag está com formatação diferente dentro do Word. Para corrigir:

  1. No Word, selecione toda a tag (ex: {{VENDEDOR_NOME}})
  2. Apague e redigite manualmente — não copie e cole
  3. Certifique-se de que toda a tag tem a mesma fonte e tamanho
  4. Salve e faça upload novamente
